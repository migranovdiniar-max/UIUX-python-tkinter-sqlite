# ?? EasyEng — Обучающее приложение по английскому языку

Простое десктопное приложение для изучения английского, с поддержкой пользователей, уровней и админ-панелью.

---

## ?? Функционал

### ? Возможности

- Вход и регистрация
- Две роли: **Студент** и **Администратор**
- Администратор регистрируется только по **секретному токену**
- Студент указывает:
  - Текущий уровень (A1 ? C2, включая A1.1, A1.2 и т.д.)
  - Цель (например, B2 ? C1)
  - Дата начала — автоматически
- Админ может:
  - Управлять пользователями
  - Просматривать уровни
  - Добавлять/редактировать темы, словарь, упражнения

---

## ?? Аутентификация

### Регистрация

- **Студент**:
  - Имя, email, пароль
  - Текущий уровень и цель
- **Администратор**:
  - Имя, email, пароль
  - **Секретный токен** (из `.env`)
  - Только при правильном токене регистрация проходит

> Пароли хранятся в **хэшированном виде** (Argon2)
